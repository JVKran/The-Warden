Detailed Description

sfml object that displays text

This object will display a string of text on a window screen

This constructor does initialize the default attributes. input is the string we will display, position is the location of the object (left upper corner), size is the size of the text itself, color is in what color the text will be displayed and centered is a bool that decides if the position is the center of the object or the left upper corner of the object (usefull if you want to center a text object in the middle of the screen with changable size)
