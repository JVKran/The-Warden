Detailed Description

Sprite Animation.

This class features an animated spritesheet. Used to easily create an animation based on a spritesheet.

Set the specified start and end frame to loop trough.

This function will set the specified start and end frame, the missingRow, the specifiedColumnsHeight and enables the specifiedFrameLoop().

Parameters
rowColumnThis has a row and a column, The row is on which sprite to begin. The column is on which column to begin.
missingRowColumnThis has a missing row and a columnHeight, The missing row is how many sprites there are left in a row. The columnHeight is to specify if the amount of frames is longer than one row.
leftThis enables the leftside of a sprite.
